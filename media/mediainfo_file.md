## MediaInfo

```bash
sudo apt install mediainfo
```

usage:
```bash
mediainfo fhs.pdf

#  General
#  Complete name                            : fhs.pdf
#  Format                                   : PDF
#  Format version                           : 1.5
#  File size                                : 100 KiB
#
#  Text
#  Format                                   : PDF
```


## File

```bash
sudo apt install file # usually preinstalled
```

usage:
```bash
file fhs.pdf

#  fhs.pdf: PDF document, version 1.5
```
