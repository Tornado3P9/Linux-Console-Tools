## vi file_name

INSERT MODE: i, I  
             a, A  
             o, O  

Exit INSERT MODE by typing the ESC key  

quit:           :q!  
quit and save:  :wq!  

quit and save shortcut:   ZZ  



To practice Vim with the help of a small tutor, type: vimtutor  
The file resets after use. Feel free to destroy the testfile.
